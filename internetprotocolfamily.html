Die Internetprotokollfamilie

Das Internet besteht aus ganz unterschiedlichen Protokollen, die dann unter diesem Namen zusammengefasst sind. Hier ein paar Beispiele:

Aufgrund der enormen Wichtigkeit von TCP und IP, wird mit TCP/IP oft die gesamte Internetprotokollfamile bezeichnet. Dazu gehören aber nicht nur TCP und IP sondern zum Beispiel auch folgende Protokolle:

  • FTP, das File Transfer Protocol
  • HTTP(S), das Hyper Text Transfer Protocol (Secured) um "Hyper Text" zu transportieren
  • IMAP, das Internet Message Access Protocol um auf eMail-Postfach zuzugreifen
  • SMTP, das Simple Message Transfer Protocol um eMails zu senden
  • SSH, das Secures Shell Protokoll, für entfernte Kommandozeile
  • UDP, das User Datagram Protocol, die Verbindungslose Variante von TCP/IP.
  • TLS, das Transmission Layer Security (ehem. SSL) für verschlüsseltes TCP
  • ICMP, das Internet Control Message Protocol
