HTML

HTML heißt "Hyper Text Markup Language", was soviel bedeutet wie Hypertext Auszeichnungssprache. HTML ist zur Strukturierung von Text vorgesehen, nicht zu dessen Formatierung, dazu wird CSS genutzt.

Genereller Aufbau einer HTML Datei


<!DOCTYPE html>
<html>
  <head>
    <title>Titel</title>
    <!-- weitere Kopfinformationen -->
    <!-- Kommentare werden nicht angezeigt. -->
  </head>
  <body>
  <!-- Inhalt ("Körper") der Webseite -->
    <p>Ein Absatz der Webseite</p>
  </body>
</html>

Generell ist HTML aus Tags aufgebaut, die Informationen einschließen. Tags sind dadurch erkennbar, dass sie von < bzw. > eingeschlossen sind. Sie schließen die Informationen ein, in zuerst einfach der Tagname zwischen < und > geschrieben wird, dann die Information folgt und dann erneut der Tagname zwischen ein < und > geschrieben wird, wobei nach dem < ein / folgt. Wenn wir das Tag "Fett" annehemen, würde z.B. <Fett>Troet</Fett< Troet fett schreiben. Mensch kann dem Tag jetzt auch noch Attribute hinzufügen, z.B. könnten wir bei unserem Beispiel angeben, wie dick es geschrieben werden soll. Das sieht dann in etwa so aus, wenn wir vom Attribut "WieFett" ausgehen: <Fett WieFett="SehrFett" >Troet<Fett>.
So sind dann alle Tags in HTML aufgebaut. Ein paar Beispieltags:

  • p: ein Absatz
  • h1, h2, h3, h4, h5, h6: eine Überschrift, h1 ist die größte, h6 die kleinste.
  • pre: Vorformatierter Text
  • input: Eingabefelder (Buttons, Textfelder, Checkboxen)
